## 📌 Sobre o Projeto

Este é um projeto simples desenvolvido com o objetivo de exercitar e testar o fluxo de desenvolvimento focado em agentes autônomos de IA na plataforma **Google Antigravity**. 

O projeto simula um painel de controle matinal unificado (Dashboard), consolidando informações do cotidiano de um desenvolvedor ou profissional, tais como:
* **Resumo de e-mails:** Contagem de mensagens não lidas e importantes.
* **Google Calendar:** Próximos eventos e reuniões do dia com contagem regressiva.
* **Google Tasks:** Monitoramento de tarefas pendentes e prazos cruciais.
* **Volumetria Semanal:** Gráfico de contexto do fluxo de e-mails recebidos nos últimos 7 dias.

---

## 🚀 Tecnologias Utilizadas

* **Google Antigravity IDE:** Plataforma e ecossistema de agentes usada no processo de concepção e escrita do código.
* **HTML5 / CSS3 / JavaScript:** Tecnologias base para a construção e estilização da interface web do dashboard.
* **GitHub Pages:** Hospedagem gratuita e publicação do ambiente estático.

---

## 🧠 Aprendizados com o Google Antigravity

A criação deste dashboard serviu como laboratório para entender a dinâmica de desenvolvimento acelerado por IA. Os principais pontos explorados foram:
* **Prompt-to-Code:** Utilização de comandos naturais para a geração ágil da estrutura da interface.
* **Orquestração de Agentes:** Compreensão de como a IDE gerencia tarefas em background enquanto o desenvolvedor atua como arquiteto de software.
* **Metodologia Ágil no Código:** Criação iterativa de componentes visuais integrando dados fictícios organizados no formato de rotina diária.

## 💻 Como Rodar Localmente

Se você deseja clonar e visualizar este projeto na sua máquina:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd calendar
   ```

3. **Abra o arquivo principal:**
   Basta abrir o arquivo `index.html` diretamente no seu navegador ou utilizar a extensão *Live Server* no seu editor de código (como o VS Code ou na própria interface do Antigravity).
